# Woodland Ways Apprentice Trainer — Planning Doc

## Purpose
A retrieval-based training system that turns instructors from 
"recognising the right answer" into confidently recalling, 
explaining, and delivering every part of the bushcraft syllabus.

## Progression System
- Spark — Recognition (multiple choice)
- Ember — Guided Recall (fill in the blank)
- Flame — Free Recall (full typed answers, spellings graded)

## Confidence Rating
After every answer the user rates themselves:
- Low / Medium / High
- High confidence + wrong = urgent review
- Low confidence + correct = reinforce soon

## Spaced Repetition
- Correct answer = delayed reappearance
- Wrong answer = rapid repetition
- Weak answers = re-tested soon

## First Module — Foraging
Starting with 3 plants:
- Stellaria media (Chickweed)
- Glechoma hederacea (Ground Ivy)
- Oxalis acetosella (Wood Sorrel)

## Content Plan
- Build Foraging module fully first
- Add plants gradually over time
- Once Foraging is solid, move to next module
